# 필수 Hook (useState, useEffect, useRef)
    - 이름이 use로 시작함
    - 제공되는 편리한 함수
    - 함수가 return 해주는 걸 구조분해 해서 보통 사용함
    - 하지만 무엇을 return 해주는 지 문서를 보지 않으면 알 수 없음!
        => 처음보는 hook이라면 꼭 문서나, 구글링 해야함
    - react.dev 가면 좋은 설명과 예제들이 나래비로 기다리고 있음!
